# 주마등 : 시네마 v5

> 6초 인생 자동 타임라인 생성기 + 흑백 감성 + 필름 그레인 + 저승사자 + 은은한 등불 애니

---

## 🔹 기능

- 일기 입력 → 자동 씬 분할 (8초 단위)
- 2시간 자동 타임라인 모드
- 캐릭터 스타일 선택 / 직접 입력 가능
- 흑백 감성 + 필름 그레인 + 저승사자 SVG
- 은은한 등불 애니메이션
- 모바일 반응형
- Grok 붙여넣기용 프롬프트 생성
- 상/하단 광고 슬롯 포함
